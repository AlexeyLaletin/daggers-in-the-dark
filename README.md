# Daggers in the Dark — карта фракций для «Клинков во Тьме»

Проектируемое приложение для мастера: интерактивная карта города с **территориями фракций**, которые можно **рисовать/перекрашивать полупрозрачными цветами** прямо поверх карты, а также система **заметок и связей** между фракциями/людьми/местами (в стиле Obsidian), режимы **GM/Player** и **таймлайн** снимков состояния города по датам.
